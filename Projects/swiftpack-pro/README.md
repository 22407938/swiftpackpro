# SwiftPack Pro Cleaning Service

A polished Next.js marketing website for SwiftPack Pro, built with TypeScript and Tailwind CSS.

## Local development

```bash
npm install
npm run dev
```

Open http://localhost:3000 to view the site.

## Production build

```bash
npm run build
```

## Deployment

This project is ready for Vercel deployment. Set the project root to this folder and use the environment value from `.env.example` if needed.
