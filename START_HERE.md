## 🚨 If You Get Stuck

**Problem** → **Solution**

| Problem | Solution |
|---------|----------|
| "Git not found" | Follow: **Install Git** section in DEPLOYMENT_GUIDE.md |
| "Deployment failed" | Check: Vercel dashboard build logs → Look for error |
| "Domain not connecting" | Wait: DNS takes 24 hrs + check: https://dnschecker.org |
| "Commands not working" | Use: Explicit Node.js path from QUICK_COMMANDS.md |
| "GitHub push fails" | Check: Username/password correct, repo exists online |
| "Site shows 404" | Verify: Domain added in Vercel Settings → Domains |
